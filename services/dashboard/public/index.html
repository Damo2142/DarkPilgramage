Echo Cookie Refresh Guide
When Echo voice stops working (cookie expired):
1. Open alexa.amazon.com in Chrome, log in
2. Press F12 → Network tab → refresh page
3. Click any request → copy the Cookie: header value
4. In Console: document.cookie.match(/csrf=([^;]+)/)?.[1]
5. SSH in, edit ~/dark-pilgrimage/co-dm/.env
6. Update ALEXA_COOKIE and ALEXA_CSRF
7. docker compose build --no-cache && docker compose up -d
